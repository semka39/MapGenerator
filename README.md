# MapGenerator

Алгоритм генерации карт высот.

![Процесс сглаживания](https://i.ibb.co/4gDFbhWR/output.gif)
![Примеры карт](https://i.ibb.co/WvhHjQwc/image.png)

## Описание алгоритма

1. Создаётся **первичная шумовая карта** - поле случайно заполненное логическими значениями.
2. Создаётся **карта высот**, заполненная случайными значениями от 0 до 1.

Далее производятся несколько итераций **сглаживания**:
  - Для каждой клетки карты высот усредняются значения с соседями по соседству Мура
  - После каждой итерации выполняется нормализация
  - Если на первичной карте клетка отмечена значением `true`, то клетки вокруг неё, отмеченные `true`, не считаются соседями

## Проблемы и возможности развития

### Возможности улучшения:
- Добавление дополнительных параметров (например, выбор между соседством Мура и фон Неймана)

### Известные проблемы:
![Изображение проблемы](https://i.ibb.co/FLtvLQNj/image.png)
- **Неравномерность карты**: некоторые участки могут казаться рваными, с "дырками"
  - Причина: плотные скопления клеток, отмеченных как `true` на первичной шумовой карте
  - Возможное решение: выполнение нескольких итераций сглаживания в конце без учёта первичной шумовой карты
